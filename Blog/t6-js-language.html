In the JavaScript language, describe all the use cases for each of: parentheses ()
1. Execute a function e.g. var result = doSomething() 2. List the parameters specific to a function e.g. function(p1, ..., pN) 3. Control the structure for the following: if, for, and while loops. Parentheses are used for grouping and containing expressions and parameters. Parentheses have multiple functions relating to functions and structures. They are used to contain a list of parameters passed to functions and control structures and they are used to group expressions to control the order of execution. Some functions have no parameters and in this case, the space between parentheses is blank.
brackets []
1. Used to specify a location within an array e.g. board.cells[2] 2. Used to create the contents of an array e.g. var myArray = ['goose', 'chicken', 'parrot'];
braces {}
1. Used when defining an object e.g. var myObject = {name: goose, age: 23}; 2. Used to define the contents of a function e.g. function alpha () {}
single quotes ''
Can be used in the same cases as double quotes 1. Identify strings e.g. name = 'goose'
double quotes ""
Can be used in the same cases as single quotes 1. Identify strings e.g. var name = "goose"
semicolon ;
A statement terminator which separates elements of the program. A statement is a complete instruction to the computer and the semicolon is used to separate instructions (this is similar to the period "." in written English).
