How does JavaScript compare to HTML and CSS?
HTML and CSS could be described as more of a two-dimensional struture. HTML is like the the bones and basic elements required of a website. CSS then adds more dimension in terms of appearance and style. Javascript is more three-dimensional, using more interactive elements with the user, such as drop-down menus, or slideshows, or hiding content on a page.
Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.
One example of control flow could be deciding what to have for breakfast, if there is bread available, you will east this for breakfast. You will continue to have bread until all of the bread is gone. If there is no bread, you will have a banana instead. An if statement would be used here to figure out what you'll have, if there is bread, you will eat the bread, or else, you will have a banana. When eating the bread each day, a loop could be used to describe this. The loop is repeated, ie you eating bread, until the condition is met. That condition would be that the bread is all gone. Hopefully this helps explain how control flow and loops could be used to describe everyday decisions.

Explain what functions are and why they are useful.
Functions are code used to perform tasks and can return values. In order for it to perform a task, it needs to be called, using a name in which it was defined. A Function only needs to be defined once, but are useful because they can be used multiple times once defined, which makes your code more efficient. They're also useful in the way that you can provide the same function with different data, which can allow it to produce different results.
