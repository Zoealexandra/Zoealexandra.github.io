What is grid based design?

A grid based design is a grid framework, that allows elements to be easily structured and added in the same manner as a typical grid. This can include using rows and columns, and using sections fo the grid for different purposes, i.e. Headers, Footers, or standout elements.

Whats all the hype about responsive webpages?

Responsive webpages are important when you think of the number of devices that people use today. Because of the variation in devices, websites need to be able to reach as many people as possible in the easiest way possible. Having a responsive website, allows a user to get the same information they would on their pc, on their mobile, and have it be just as easy to understand and navigate.

What is semantic structure?

Semantic structure is where the html describes the meaning of the content it contains. For example, the 'header' tag is used at the beginning of a site, as this 'heads' the page.
